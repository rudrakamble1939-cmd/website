# FashionHub - React E-Commerce Homepage

A modern and responsive E-Commerce homepage built using React.js.

## Features

* Responsive Navbar
* Hero Section
* Categories Section
* Trending Products
* Customer Testimonials
* Footer

## Technologies Used

* React.js
* JSX
* CSS3
